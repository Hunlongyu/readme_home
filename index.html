ReadMe

一些 Github 库会在 README.md 里介绍使用方法和详情。该软件则可以收集、整理、查看 README 文件。

PWA 网页版 Win 桌面版

产品特色

快速添加

可以通过模糊搜索和精确搜索，快速找到项目的README文件。

标签分类

通过单个标签的方式进行分类，通过标签即可快速找到项目。

快速预览

点击即可预览MD文件，支持MD里代码高亮。

多重选择

有Exe版本的软件，也有PWA小体积的网页版。

PWA使用方法

PWA网页版除了收藏网址使用。还可以添加到桌面，类似小程序。Chrome浏览器打开PWA网页版。点击菜单，选择安装"ReadMe"，即可在桌面看到ReadMe图标。

搜索添加页面

默认打开页面，点击左侧的搜索图标即可进入搜索页面，输入项目的名称，或者完整的作者名字/项目名称。

MD预览页面

点击中间列表里的数据，右侧则可以快速预览MD文件，同时可以快速修改，和删除。

标签分类页面

列出了所有的标签，统计了每个标签共有多少条数据。同时可以快速预览、修改和删除。
